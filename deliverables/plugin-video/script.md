# Great Minds Plugin - Video Script (45s)

**Target audience:** Developers, AI builders, Claude Code power users
**Tone:** Sharp, propulsive, Sorkin-style walk-and-talk energy
**Format:** Remotion (React-based video framework)

---

## Scene 1 (0:00 - 0:05)
**Visual:** Dark screen. A single line types itself into a terminal: `$ npx plugins add sethshoultes/great-minds-plugin`. The cursor hangs. Then -- a cascade. Fourteen process lines ignite down the left margin, each tagged with a name: Jobs. Musk. Hamilton. Lovelace. Huang...
**Audio/VO:** "One command. That's it. One command and you've hired fourteen people who never sleep, never bill hourly, and never ask if this meeting could've been an email."
**Motion:** Terminal text at 35ms per character. Agent names cascade in at 80ms intervals, each pulsing once on arrival.

---

## Scene 2 (0:05 - 0:15)
**Visual:** A split view. Left panel: a tmux grid, eight panes, all frozen. Red text: `0 of 8 dispatched`. Right panel: the same task list, but now lines branch into worktrees -- parallel file trees fanning out like a subway map. Green checkmarks ripple through. Counter climbs: `12... 18... 25+ tasks complete`.
**Audio/VO:** "We tried tmux dispatch first. Zero successes. Not one. So we threw it out and built on the Agent tool with worktrees. Twenty-five completed tasks and counting. You don't defend the architecture that failed. You defend the users who need it to work."
**Motion:** Left panel flickers and dims. Right panel expands to fill the frame. Checkmarks cascade with a 60ms stagger.

---

## Scene 3 (0:15 - 0:28)
**Visual:** An org chart assembles itself. At the top: a "Board of Directors" node. Below it, three tiers materialize -- Strategy, Build, Operations. Twelve skill badges orbit the agents like electrons: `brand-audit`, `market-fit`, `code-review`, `qa-sweep`... Decoupled cron pulses radiate from a HEARTBEAT node at the bottom, keeping everything in rhythm.
**Audio/VO:** "Fourteen agents. Twelve skills. A board of directors that actually directs. Decoupled crons so nothing stalls waiting on something it doesn't need. GSD integration so your tasks don't live in a doc somewhere -- they live in the system that executes them."
**Motion:** Org chart builds top-down with 120ms per tier. Skill badges float in with slight orbital drift. Cron pulses are concentric rings at 2-second intervals.

---

## Scene 4 (0:28 - 0:38)
**Visual:** A terminal session. The user types a PRD filename. The system responds: `Board convened. Strategy locked. Build phase initiated.` Below that, a live progress feed -- agents checking in, deliverables landing in `/deliverables/final/`. A browser opens. A finished site.
**Audio/VO:** "You give it a problem. It convenes a board. The board debates -- and debate is the product, because consensus is just the absence of someone brave enough to disagree. Then they build. Then they ship. While you were asleep."
**Motion:** Terminal lines appear at natural typing speed. Progress feed scrolls with real agent names. Browser window scales up to fill frame over 1.5 seconds.

---

## Scene 5 (0:38 - 0:45)
**Visual:** Black. The install command returns, centered, in monospace: `npx plugins add sethshoultes/great-minds-plugin`. Below it, a single line in a smaller weight: `One install. A full AI agency.`
**Audio/VO:** "Great Minds plugin. One install. A full AI agency."
**Motion:** Command fades in over 400ms. Tagline fades in 600ms later. Hold for 3 seconds. Cut to black.

---

## Production Notes

- **Total runtime:** 45 seconds (target)
- **Music:** Percussive minimal electronic -- think Trent Reznor and Atticus Ross scoring a product launch. Tension through Scene 2, opens up in Scene 3, resolves cleanly on the CTA.
- **Typography:** JetBrains Mono for all terminal/code. Inter for UI and tagline. No serif anywhere.
- **Color palette:** Near-black background (#09090b), electric blue for active agents (#3b82f6), failure red for tmux (#ef4444), success green for worktrees (#22c55e), warm amber for cron pulses (#f59e0b).
- **Pacing note:** This is a Sorkin script. The VO never stops moving. Every sentence hands off to the next like a relay baton. If the viewer catches their breath, you've lost them.
- **Key differentiator from Shipyard video:** Shipyard sells the outcome (your site, built overnight). This sells the tool (the plugin itself, the architecture, the lesson learned). The Shipyard video is a promise. This one is a proof.
