# Great Minds Plugin - Video Script (50s)

**Target audience:** Developers, AI builders, Claude Code power users
**Tone:** Sharp, propulsive, confident
**Format:** Remotion (React-based video framework)

---

## Scene 1 (0:00 - 0:12)
**Visual:** Dark screen. A question fades in large: "What if your IDE had a team?" Beat. Then a terminal appears. A single line types: `$ npx plugins add sethshoultes/great-minds-plugin`. The cursor hangs. Then — a cascade. Fourteen agent names ignite down the left margin: Jobs. Musk. Hamilton. Huang. Winfrey. Buffett. Rhimes...
**Audio/VO:** "What if your IDE had a team? Not copilots. Not autocomplete. A team. Fourteen minds — each with a role, a perspective, and a reason to disagree with the others. One command. That's all it takes."
**Motion:** Question fades in centered over 600ms. Terminal types at 35ms per character. Agent names cascade at 80ms intervals, each pulsing once.

---

## Scene 2 (0:12 - 0:28)
**Visual:** An org chart assembles itself. At the top: "Board of Directors" — Jensen Huang, Oprah Winfrey, Warren Buffett, Shonda Rhimes. Below: three tiers — Strategy, Build, Operations. Twelve skill badges orbit the agents. Decoupled cron pulses radiate from a HEARTBEAT node, keeping everything in rhythm.
**Audio/VO:** "A board of directors that actually directs. Jensen asks where the moat is. Oprah asks if a real person would understand it. Buffett asks if it's a business or a hobby. Fourteen agents. Twelve skills. Decoupled crons so nothing stalls. GSD integration so tasks execute, not just exist."
**Motion:** Org chart builds top-down at 120ms per tier. Skill badges float in with slight orbital drift. Cron pulses are concentric rings at 2-second intervals.

---

## Scene 3 (0:28 - 0:42)
**Visual:** A terminal session. The user types a PRD filename. The system responds: `Board convened. Strategy locked. Build phase initiated.` Below, a live progress feed — agents checking in, deliverables landing. A browser opens. A finished site.
**Audio/VO:** "You give it a problem. It convenes a board. The board debates — and debate is the product, because consensus is just the absence of someone brave enough to disagree. Then they build. In parallel. On isolated worktrees. Then they ship. While you were asleep."
**Motion:** Terminal lines at natural typing speed. Progress feed scrolls with real agent names. Browser window scales up to fill frame over 1.5 seconds.

---

## Scene 4 (0:42 - 0:50)
**Visual:** Black. The install command returns, centered: `npx plugins add sethshoultes/great-minds-plugin`. Below: `One install. A full AI agency.` Below that: `greatminds.company/work/great-minds-plugin`
**Audio/VO:** "Great Minds plugin. One install. A full AI agency."
**Motion:** Command fades in over 400ms. Tagline 600ms later. URL types out. Hold 3 seconds. Cut to black.

---

## Production Notes

- **Total runtime:** 50 seconds
- **Music:** Percussive minimal electronic. Tension builds through Scene 2, opens in Scene 3, resolves on CTA.
- **Typography:** JetBrains Mono for terminal/code. Inter for UI and tagline.
- **Color palette:** Near-black (#09090b), purple accent (#a855f7), success green (#22c55e), warm amber for crons (#f59e0b).
- **Key differentiator:** This sells the tool itself — the plugin, the architecture, the team you install. Not the outcome, the system.
