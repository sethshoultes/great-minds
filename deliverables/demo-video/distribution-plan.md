## Embedding on Landing Page

### Technical Requirements
- Video hosted on YouTube (unlisted) or direct MP4
- Autoplay: OFF (user controls)
- Muted: OFF (sound is important)
- Loop: OFF
- Poster/thumbnail: First frame or custom
- Position: Hero section, above fold

### Embed Code (YouTube)
```html
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/[VIDEO_ID]"
  title="PIPELINE Demo"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>
```

3. Consider paid promotion boost
4. Analyze retention data for drop-off points

### If landing page has issues:
1. **Do NOT post** until fixed
2. Use YouTube link directly as fallback
3. Add CTA to video description
