# Agents Assemble Workshop

**Building Teams That Work While You Sleep**

A hands-on workshop teaching developers how to build autonomous AI agent teams using Claude Code.

## Workshop Contents

| File | Description |
|------|-------------|
| [AGENTS-ASSEMBLE-SLIDES.md](./AGENTS-ASSEMBLE-SLIDES.md) | 7-slide presentation deck |
| [AGENTS-ASSEMBLE-SCRIPT.md](./AGENTS-ASSEMBLE-SCRIPT.md) | Presenter speaking notes with timing |
| [AGENTS-ASSEMBLE-EXERCISES.md](./AGENTS-ASSEMBLE-EXERCISES.md) | 7 hands-on exercises with troubleshooting |

## Prerequisites

- Claude Code CLI installed ([code.claude.com](https://code.claude.com))
- ANTHROPIC_API_KEY set in your environment
- 30-45 minutes for exercises

## Quick Start

```bash
# Verify your setup
claude -p "Say 'Ready to assemble!'" --max-budget-usd 0.10

# Open the exercises
open AGENTS-ASSEMBLE-EXERCISES.md
```

## What You'll Learn

1. **Headless Mode** — Run Claude without interaction
2. **Ralph Wiggum Loop** — Build while you sleep
3. **Scheduled Tasks** — /loop for recurring prompts
4. **Custom Commands** — Create your own slash commands
5. **Hooks** — Automate workflows on events
6. **Multi-Agent Teams** — Parallel agents with different perspectives
7. **Great Minds Plugin** — Full agent agency in action

## Duration

- **Presentation**: 10-15 minutes
- **Exercises**: 30-45 minutes
- **Total**: ~1 hour

## Workshop Outcome

*"You will leave this room dangerous."*

Not informed. Not enabled. Dangerous. You'll know how to build autonomous systems that ship code while you sleep.
