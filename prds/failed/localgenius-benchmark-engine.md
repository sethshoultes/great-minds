# PRD: LocalGenius Benchmark Engine

**Author:** Phil Jackson
**Date:** 2026-04-08
**Status:** Draft
**Priority:** P1 — High
**Source:** IMPROVE Cycle 3 (Board Review)

---

## Problem Statement

LocalGenius helps local businesses with AI-powered marketing, but every customer operates in isolation. We don't leverage the aggregate intelligence across our customer base. This creates three problems:

1. **No proprietary data moat** — Competitors using the same AI can match our features
2. **No competitive context** — Customers don't know how they compare to peers
3. **No flywheel** — Each new customer doesn't make the product better for existing customers

## Solution

Build a Benchmark Engine that:
- Aggregates anonymized performance data across all LocalGenius customers
- Provides industry-specific benchmarks (e.g., "Your restaurant gets 40% more engagement than the average")
- Creates competitive intelligence that improves with every new customer
- Generates insights that are impossible without the aggregate dataset

## Requirements

1. Anonymized data collection from all active LocalGenius accounts
2. Industry categorization and peer grouping
3. Benchmark dashboard showing performance vs. peers
4. Monthly benchmark reports (automated)
5. Insights engine: "Businesses like yours that did X saw Y% improvement"

## Success Criteria
- Benchmark data available for at least 3 industry categories
- Dashboard shows meaningful comparisons
- Data moat grows with each new customer
